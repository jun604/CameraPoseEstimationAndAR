# AR with Camera Calibration & Pose Estimation
OpenCV를 이용하여 카메라의 파라미터를 산출하고, 체스보드 기준의 3차원 포즈(Pose)를 추정하여 가상 객체를 증강하는 프로그램</br>
solvePnP 알고리즘을 통해 카메라의 위치와 회전을 계산하고, projectPoints로 3D 도형을 이미지 평면에 투영하도록 구현</br>
학습 이미지 선택 기능, 실시간 카메라 좌표 출력, 가상 3D 구조물 렌더링 및 영상 저장 지원

## 기능
1. 영상 전처리 및 해상도 최적화
   - 입력 비디오를 가로 800px(TW) 기준으로 리사이즈하여 일관된 처리 환경 제공
   - 원본 영상의 종횡비를 유지하여 좌표 계산의 정확도 확보
2. 인터랙티브 이미지 선택 (Calibration Data)
   - 사용자 개입을 통해 체스보드가 명확히 검출되는 프레임만 선별 수집
   - Space(일시정지): 코너 검출 상태를 시각적으로 확인
   - Enter(이미지 선택): 현재 프레임을 데이터셋에 추가하고 다음 장면으로 이동
   - 데이터 수집 과정 전반을 영상 파일로 자동 기록
3. 정밀 카메라 캘리브레이션
   - 선택된 프레임들을 분석하여 카메라 행렬 K (초점거리(fx, fy), 주점(cx, cy)) 및 왜곡 계수 산출
   - 3D 공간의 체스보드 좌표와 2D 이미지 좌표 간의 대응 관계를 학습
4. 실시간 포즈 추정 및 AR 시각화
   - solvePnP: 카메라 매트릭스를 기반으로 체스보드 평면에 대한 카메라의 상대적 위치(tvec)와 방향(rvec) 추정
   - 3D 객체 투영: 체스보드 위에 가상의 박스 및 구조물을 배치하고 프레임별로 렌더링
   - 위치 데이터 출력: 실시간으로 계산된 카메라의 세계 좌표(XYZ)를 화면 상단에 표시
5. 다중 출력 자동 저장
   - 작업 완료 시 타임스탬프가 포함된 두 가지 형식의 영상 저장
     + Play_파일명_날짜.avi: 사용자가 보정 데이터를 선택하는 전 과정 기록
     + Result_파일명_날짜.avi: 실시간 포즈 추정 및 3D 객체가 증강된 결과 영상

### 조작 방법
- 이미지 선택 단계 (Calibration)
  + Space : 영상 일시정지 및 체스보드 코너 표시
  + Enter : (일시정지 상태에서) 현재 프레임 선택 및 재생 재개
  + ESC : 이미지 선택 종료 및 캘리브레이션/포즈 추정 시작
- 포즈 추정 단계 (Pose Estimation)
  + Space : 일시정지
  + ESC : 프로그램 종료 및 리소스 해제
